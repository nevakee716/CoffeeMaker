This feature allow to add some check and automatically fill values when editing an object :
- Check number of associations with a minimum and/or maximum
- Automatically filled value depending of other properties and associations
 Questa funzione consente di aggiungere un controllo e riempire automaticamente i valori durante la modifica di un oggetto:
- Controlla il numero di associazioni con un minimo e/o massimo
- Valore riempito automaticamente in base ad altre proprietà e associazioni
Cette fonctionnalité permet d’ajouter des vérifications et de remplir automatiquement les valeurs lors de la modification d’un objet:
- Vérifier le nombre d'associations avec un minimum et/ou un maximum
- Valeur automatiquement renseignée en fonction d'autres propriétés et associations
