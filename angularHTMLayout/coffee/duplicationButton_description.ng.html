This feature add a copy button inside objectPages, when clicking on it, it will duplicate the object and his associations and the properties of association.
The newly created object can also be automatically associated to the original object.
In order to see the duplicate button, an cwUser must be able to create the objectType.
The cwUser also need to be an approuver of the view or the view need to have no approuvers.
