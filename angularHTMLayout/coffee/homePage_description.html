This feature allow you to do customize your home Page by :
- Disabling the my menu feature
- Adding a list of the last modified objects
- Adding the description of Objects on the homePage
 Questa funzione ti consente di personalizzare la tua Home Page da:
- Disabilitare la funzione del mio menu
- Aggiunta di un elenco degli ultimi oggetti modificati
- Aggiunta della descrizione degli oggetti nella pagina iniziale
Cette fonctionnalité vous permet de personnaliser votre page d'accueil par:
- Désactivation de la fonctionnalité Mon menu
- Ajout d’une liste des derniers objets modifiés
- Ajout de la description des objets sur la page d'accueil
